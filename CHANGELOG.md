# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-06-29
### Added
- Google Drive Sync & Backup feature.
- OAuth 2.0 PKCE implementation for secure, backend-less client-side authentication.
- Integration with `drive.appdata` (App Data Folder) for private, application-specific backups.
- Manual "Backup Now" and "Restore" functionality in the UI.
- New "Sync" tab in the bottom navigation bar.

## [1.0.0] - 2026-06-28
### Added
- Initial release.
- Income and expense tracking with daily breakdowns.
- Subscription management with monthly payment tracking and installment support.
- Category-based spending visualization with charts.
- PWA support for offline use.
